## 六、动效设计

### 6.1 核心动效

| 动效名称 | 触发场景 | 动画时长 | 说明 |
|---------|---------|---------|------|
| 消息滑入 | 新消息出现 | 300ms | 从下往上淡入 |
| 加载脉动 | AI思考中 | 持续 | 呼吸灯效果 |
| 按钮反馈 | 按钮点击 | 100ms | 缩放+高亮 |
| 进度更新 | 知识点完成 | 500ms | 进度条动画增长 |
| 公式淡入 | 公式展示 | 400ms | 淡入显示 |

### 6.2 加载动画

```css
@keyframes pulse {
  0% { opacity: 0.4; }
  50% { opacity: 1; }
  100% { opacity: 0.4; }
}

.loading-dot {
  animation: pulse 1.5s infinite;
}

.loading-dot:nth-child(2) {
  animation-delay: 0.3s;
}

.loading-dot:nth-child(3) {
  animation-delay: 0.6s;
}
```
