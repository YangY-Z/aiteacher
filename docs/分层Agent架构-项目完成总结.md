## 🚀 API文档

### 端点列表

#### 1. 启动教学(工具增强)

```http
POST /api/v1/teaching-v2/session/{id}/teach-v2?use_tools=true
```

**参数**:
- `use_tools`: 是否启用工具增强 (默认true)

**响应**: SSE事件流

**示例**:
```javascript
const eventSource = new EventSource(
  '/api/v1/teaching-v2/session/123/teach-v2?use_tools=true'
);

eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log(data);
};
```

#### 2. 查看可用工具

```http
GET /api/v1/teaching-v2/session/{id}/tools/available
```

**响应**:
```json
{
  "tools": ["image_generation"],
  "session_id": "123",
  "knowledge_point": "K1"
}
```

#### 3. 获取图片

```http
GET /api/v1/teaching-v2/images/{image_id}
```

**响应**:
```json
{
  "id": "IMG_K1_001",
  "title": "正比例函数y=2x的图像",
  "url": "/static/images/IMG_K1_001.png",
  "description": "展示正比例函数的图像特征"
}
```

#### 4. 生成图片

```http
POST /api/v1/teaching-v2/images/generate
```

**请求**:
```json
{
  "concept": "函数图像",
  "type": "example",
  "knowledge_point_id": "K3"
}
```

**响应**:
```json
{
  "id": "IMG_GENERATED_001",
  "source": "template",
  "url": "/static/images/generated/IMG_001.png",
  "cost": 0.02
}
```
