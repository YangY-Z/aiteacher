## 🎯 使用指南

### 1. 启动前端

```bash
cd ai-teacher-frontend
npm install
npm run dev
```

### 2. 启动后端

```bash
cd ai-teacher-backend
python3 -m uvicorn app.main:app --reload
```

### 3. 测试流程

1. 登录系统
2. 进入学习页面
3. 默认启用"工具增强"模式
4. 开始学习，观察图片显示
5. 切换到"标准模式"，验证无图片

```tsx
<Button 
  type={state.useTools ? 'primary' : 'default'}

</Button>
```

## 🔧 技术细节

### 1. 图片加载流程

```
后端生成图片 → 返回image_id → 
前端调用getImage API → 
缓存到Hook → 渲染组件
```

### 2. 事件处理流程

```
segment事件(含image_id) → 
添加到消息列表 → 
TeachingImage组件自动加载图片 → 
显示给用户
```

### 3. 新旧模式切换

```
useTools=true → teachingV2Api (工具增强)
useTools=false → learningApi (标准模式)
```
