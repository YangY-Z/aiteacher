# 年级科目配置功能说明

## 功能概述

在课程配置系统中新增了**基础配置**标签页,用于管理年级和科目。

## 访问方式

1. 进入课程配置页面
2. 点击顶部的"基础配置"标签页
3. 即可看到年级和科目管理界面

## 功能特性

### 1. 年级管理

**查看年级列表**
- 显示所有已配置的年级
- 每个年级以标签形式展示

**添加年级**
- 点击"添加年级"按钮
- 输入年级名称(如:初一、初二等)
- 点击"添加"完成

**删除年级**
- 点击年级标签右侧的关闭按钮
- 确认后删除

### 2. 科目管理

**查看科目列表**
- 显示所有已配置的科目
- 每个科目以标签形式展示

**添加科目**
- 点击"添加科目"按钮
- 输入科目名称(如:数学、语文等)
- 点击"添加"完成

**删除科目**
- 点击科目标签右侧的关闭按钮
- 确认后删除

### 3. 数据持久化

配置会自动保存到浏览器 localStorage 中:
- `admin_grades`: 年级配置
- `admin_subjects`: 科目配置

**注意**: 清除浏览器缓存会导致配置丢失

## 使用场景

### 场景一: 创建新章节时选择年级和科目

1. 在"章节管理"标签页点击"新建章节"
2. 年级和科目的下拉列表会自动显示已配置的选项
3. 选择后创建章节

### 场景二: 自定义年级和科目

1. 进入"基础配置"标签页
2. 添加或删除年级/科目
3. 点击"保存配置"按钮
4. 返回"章节管理"使用新配置

## 默认配置

系统预设了常用的年级和科目:

**年级**
- 初一、初二、初三
- 高一、高二、高三

**科目**
- 数学、语文、英语
- 物理、化学

## 数据流程

```
用户配置年级/科目
    ↓
保存到 localStorage
    ↓
章节管理页面读取配置
    ↓
下拉列表显示选项
    ↓
创建章节时使用
```

- 采用卡片式布局
- 圆角标签设计
- 统一的交互反馈

### 用户友好

- 可视化管理
- 即时保存
- 防重复添加

- 使用 Modal 组件实现添加弹窗

### 状态管理

```typescript
const [grades, setGrades] = useState<string[]>([]);
const [subjects, setSubjects] = useState<string[]>([]);
const [newItem, setNewItem] = useState('');
```

### 持久化

```typescript
// 保存
localStorage.setItem('admin_grades', JSON.stringify(grades));
localStorage.setItem('admin_subjects', JSON.stringify(subjects));

// 加载
const savedGrades = localStorage.getItem('admin_grades');
if (savedGrades) setGrades(JSON.parse(savedGrades));
```
