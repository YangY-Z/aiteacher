## 🚀 5分钟快速启动

### 第一步：启动后端 (2分钟)

```bash
# 1. 进入后端目录
cd ai-teacher-backend

# 2. 安装依赖（如果还没安装）
pip3 install -r requirements.txt

# 3. 启动后端服务
python3 -m uvicorn app.main:app --reload --port 8000
```

**验证**: 访问 http://localhost:8000/docs 看到API文档

---

### 第二步：启动前端 (2分钟)

```bash
# 1. 新终端，进入前端目录
cd ai-teacher-frontend

# 2. 安装依赖（如果还没安装）
npm install

# 3. 启动前端服务
npm run dev
```

**验证**: 访问 http://localhost:5173 看到登录页面

---

## 🐛 常见问题

### 问题1: 图片加载失败

**现象**: 图片显示错误提示

**原因**:
- 后端未启动
- 图片ID不存在
- 网络问题

**解决**:
```bash
# 1. 检查后端日志
tail -f ai-teacher-backend/app.log

# 2. 手动测试API
curl http://localhost:8000/api/v1/teaching-v2/images/test_id

# 3. 检查数据库
sqlite3 ai-teacher-backend/database/teaching.db
```

---

### 问题2: 工具增强按钮无响应

**现象**: 点击按钮没有反应

**原因**:
- React状态未更新
- 浏览器缓存

**解决**:
```bash
# 1. 清除浏览器缓存
# 2. 刷新页面（Cmd+Shift+R）
# 3. 检查控制台错误
```

---

### 问题3: API调用错误

**现象**: Network显示404或500错误

**原因**:
- API路径错误
- 后端路由未注册

**解决**:
```bash
# 1. 检查API文档
open http://localhost:8000/docs

# 2. 查看后端路由
grep -r "teaching-v2" ai-teacher-backend/app/api/

# 3. 重启后端服务
```
