# 新建章节 - 操作指南

## 📍 新的位置

**"新建章节"按钮**现在位于展开面板的年级标题旁边，更符合使用逻辑。

## 🎯 操作流程

### 方式一：从年级展开面板创建

1. **点击年级卡片**
   - 在"章节管理"标签页
   - 点击任意年级的卡片（如：初二）

2. **展开详情面板**
   - 面板顶部显示年级标题
   - 标题右侧有蓝色的"新建章节"按钮

3. **点击"新建章节"**
   - 年级字段**自动填充**当前年级
   - 无需手动选择年级

4. **填写章节信息**
   - 年级：已自动填充
   - 版本：选择教材版本（人教版/北师大版等）
   - 科目：选择科目
   - 章节名称：输入章节名
   - 描述：可选
   - 预估学时：可选

### 方式二：空状态时创建

如果系统中还没有任何章节：

1. 页面显示"暂无章节"
2. 点击"创建章节"按钮
3. 需要手动选择年级

## 📋 完整操作示例

### 示例：为初二添加"一次函数"章节

1. 点击"初二"卡片
2. 展开面板显示：
   ```
   初二          [+ 新建章节] [×]
   ```
3. 点击"新建章节"
4. 表单已填充：
   ```
   年级: 初二 ✓ (自动填充)
   ```
5. 选择其他信息：
   ```
   教材版本: 人教版
   科目: 数学
   章节名称: 一次函数
   描述: 初二数学一次函数单元
   预估学时: 12
   ```
6. 点击"确定"完成创建

## 💡 提示

### 智能预填充规则
- 从年级面板打开：年级自动填充
- 从空状态打开：年级需手动选择
- 编辑现有章节：所有字段自动填充

### 最佳实践
1. **先查看年级**：确保在正确的年级下创建
2. **填写完整信息**：包括版本、科目、描述
3. **预估学时**：帮助后续排课规划

## 🔍 相关功能

### 基础配置
在"基础配置"标签页可以：
- 添加/删除年级
- 添加/删除科目
- 这些选项会出现在章节表单中

### 知识点配置
创建章节后：
- 点击章节的"配置"按钮
- 进入知识点管理页面
- 为章节添加知识点
