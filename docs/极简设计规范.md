# 极简设计规范

## 设计理念

遵循乔布斯的设计哲学: **简约而不简单**

> "简单是终极的复杂。" - Leonardo da Vinci
> 
> "要把一样东西做好,必须先把它简化。" - Steve Jobs

## 核心原则

### 1. 极简主义 (Minimalism)

**去除所有不必要的元素,只保留核心功能**

- ❌ 移除复杂的3D书籍效果
- ❌ 移除木质纹理和阴影
- ❌ 移除过多的装饰线条
- ✅ 保留清晰的卡片式布局
- ✅ 保留核心的交互功能

### 2. 大量留白 (White Space)

**让内容呼吸,给用户视觉休息的空间**

- 页面边距: `48px`
- 卡片内边距: `32px`
- 区块间距: `80px`
- 元素间距: `24px`

### 3. 清晰的层次 (Clear Hierarchy)

**通过字体大小和颜色建立视觉层次**

```
大标题: 40px / #1d1d1f / 600
副标题: 17px / #86868b / 400
卡片标题: 21px / #1d1d1f / 600
正文: 15px / #1d1d1f / 400
辅助文字: 13px / #86868b / 400
```

### 4. 优雅的过渡 (Elegant Transitions)

**所有动画都使用苹果风格的缓动函数**

```css
transition: all 0.4s cubic-bezier(0.25, 0.1, 0.25, 1);
```

### 5. 精致的颜色 (Refined Colors)

**灵感来自 Apple 官网**

- 背景: `#f5f5f7` (浅灰)
- 主色: `#1d1d1f` (深灰)
- 辅助: `#86868b` (中灰)
- 强调: `#0071e3` (苹果蓝)
- 边框: `rgba(0, 0, 0, 0.1)` (半透明)

## 组件设计

### Header 导航栏

```css
- 高度: 52px (比传统64px更精简)
- 背景: 半透明毛玻璃效果
- 阴影: 仅底部分割线
- 字体: 17px / 600
```

### Card 卡片

```css
- 圆角: 18px (更柔和)
- 阴影: 极浅的阴影
- 背景: 纯白
- 悬停: 轻微上浮 + 彩色顶部边框
```

### Button 按钮

```css
- 圆角: 20px (胶囊形状)
- 高度: 32px / 40px
- 字体: 13px / 15px
- 过渡: 0.3s ease
```

### Modal 弹窗

```css
- 圆角: 20px
- 内边距: 32px
- 动画: 缩放 + 淡入
- 遮罩: 毛玻璃模糊
```

## 交互设计

### 悬停效果

1. **卡片**: 向上浮动4px + 浅色阴影
2. **按钮**: 背景色加深
3. **列表项**: 背景色变化

### 点击效果

1. **卡片**: 展开详情面板
2. **遮罩**: 点击关闭
3. **阻止冒泡**: 内容区域

### 过渡动画

- 持续时间: `0.3s - 0.4s`
- 缓动函数: `cubic-bezier(0.25, 0.1, 0.25, 1)`
- 属性: `transform, opacity, box-shadow`

## 排版规范

### 字体家族

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 
             'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```

### 字重使用

- **600 (Semibold)**: 标题、强调
- **500 (Medium)**: 次级标题、标签
- **400 (Regular)**: 正文、辅助文字

### 行高

- 标题: `1.2`
- 正文: `1.5`
- 辅助文字: `1.4`

## 响应式设计

### 断点

- 桌面: `> 768px`
- 移动: `<= 768px`

### 移动端适配

- 减少边距: `48px → 24px`
- 单列布局: `grid-template-columns: 1fr`
- 隐藏侧边栏
- 堆叠导航

## 无障碍设计

### 颜色对比度

- 主文字: `#1d1d1f` on `#ffffff` → 16.1:1 ✓
- 辅助文字: `#86868b` on `#ffffff` → 4.5:1 ✓
- 链接: `#0066cc` on `#ffffff` → 4.8:1 ✓

### 可聚焦元素

- 所有交互元素都有 `focus` 状态
- 使用系统默认的聚焦环
- 可键盘导航

## 性能优化

### CSS

- 使用 `transform` 代替 `top/left`
- 避免复杂的 `box-shadow`
- 使用 `will-change` 优化动画

### React

- 合理使用 `useState` 和 `useEffect`
- 避免不必要的重渲染
- 使用 CSS 过渡代替 JS 动画

## 设计对比

### 之前 (Before)

- ❌ 复杂的3D书籍效果
- ❌ 木质纹理背景
- ❌ 过多的装饰元素
- ❌ 不统一的间距
- ❌ 过重的阴影

### 现在 (After)

- ✅ 简洁的卡片布局
- ✅ 浅色纯净背景
- ✅ 适度的留白
- ✅ 统一的设计语言
- ✅ 轻盈的视觉效果
