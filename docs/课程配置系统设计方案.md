## 一、需求概述

### 1.1 功能目标

实现一个**多层级课程配置系统**，支持管理员对课程体系进行可视化管理，包括：

- **年级配置**：初一、初二、初三、高一、高二、高三
- **版本配置**：人教版、北师大版、苏教版、鲁教版等
- **科目配置**：数学、语文、英语、物理、化学
- **章节配置**：课程单元划分（如"一次函数"、"二次函数"）
- **知识点配置**：具体知识点的详细设置
- **层级配置**：知识点难度层级（0-6）

### 1.2 用户角色

- **管理员**：拥有完整的课程配置权限
- **学生**：仅能查看和学习已配置的课程

## 二、数据模型设计

## 三、数据库Schema设计

### 3.1 PostgreSQL 表结构

#### 3.1.1 章节表 (chapters)

```sql
CREATE TABLE chapters (
    id VARCHAR(50) PRIMARY KEY COMMENT '章节ID，如 CH_MATH_7_REN_01',
    name VARCHAR(100) NOT NULL COMMENT '章节名称',
    grade VARCHAR(20) NOT NULL COMMENT '年级',
    edition VARCHAR(50) NOT NULL COMMENT '教材版本',
    subject VARCHAR(20) NOT NULL COMMENT '科目',
    description TEXT COMMENT '章节描述',
    sort_order INTEGER DEFAULT 0 COMMENT '排序权重',
    total_knowledge_points INTEGER DEFAULT 0 COMMENT '知识点总数',
    estimated_hours DECIMAL(5,2) COMMENT '预估学习时长（小时）',
    level_descriptions JSONB COMMENT '层级描述JSON',
    status VARCHAR(20) DEFAULT 'active' COMMENT '状态：active/draft/archived',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    CONSTRAINT chk_grade CHECK (grade IN ('初一', '初二', '初三', '高一', '高二', '高三')),
    CONSTRAINT chk_edition CHECK (edition IN ('人教版', '北师大版', '苏教版', '鲁教版', '华师大版', '人教版新教材')),
    CONSTRAINT chk_subject CHECK (subject IN ('数学', '语文', '英语', '物理', '化学', '生物', '历史', '地理'))
);

-- 索引
CREATE INDEX idx_chapters_grade ON chapters(grade);
CREATE INDEX idx_chapters_edition ON chapters(edition);
CREATE INDEX idx_chapters_subject ON chapters(subject);
CREATE INDEX idx_chapters_status ON chapters(status);
CREATE UNIQUE INDEX idx_chapters_unique ON chapters(grade, edition, subject, name);

COMMENT ON TABLE chapters IS '章节信息表';
```

#### 3.1.2 知识点表 (knowledge_points) - 修改版

```sql
CREATE TABLE knowledge_points (
    id VARCHAR(20) PRIMARY KEY COMMENT '知识点ID，如 K1, K2',
    chapter_id VARCHAR(50) NOT NULL COMMENT '所属章节ID',
    name VARCHAR(100) NOT NULL COMMENT '知识点名称',
    type VARCHAR(20) NOT NULL COMMENT '类型：概念/公式/技能',
    description TEXT COMMENT '知识点描述',
    level INTEGER NOT NULL COMMENT '层级：0-6',
    sort_order INTEGER DEFAULT 0 COMMENT '同层级内排序',
    key_points JSONB COMMENT '核心要点数组',
    mastery_criteria JSONB NOT NULL COMMENT '掌握标准配置',
    teaching_config JSONB NOT NULL COMMENT '教学配置',
    content_template TEXT COMMENT '教学内容模板（可选）',
    status VARCHAR(20) DEFAULT 'active' COMMENT '状态：active/draft/archived',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    CONSTRAINT fk_kp_chapter FOREIGN KEY (chapter_id) REFERENCES chapters(id) ON DELETE CASCADE,
    CONSTRAINT chk_kp_type CHECK (type IN ('概念', '公式', '技能'))
);

-- 索引
CREATE INDEX idx_kp_chapter ON knowledge_points(chapter_id);
CREATE INDEX idx_kp_level ON knowledge_points(level);
CREATE INDEX idx_kp_type ON knowledge_points(type);
CREATE INDEX idx_kp_status ON knowledge_points(status);

COMMENT ON TABLE knowledge_points IS '知识点表';
```

## 七、数据持久化方案

### 7.1 内存数据库扩展

```python
# 在 InMemoryDatabase 类中添加章节存储
@dataclass
class InMemoryDatabase:
    # ... 现有字段
    
    # 章节存储
    _chapters: dict[str, Chapter] = field(default_factory=dict)
    
    # 知识点存储（修改为按章节组织）
    _knowledge_points: dict[str, KnowledgePoint] = field(default_factory=dict)
    
    def save_chapters_to_file(self, file_path: str = "data/chapters.json"):
        """保存章节数据到JSON文件"""
        data = {
            "chapters": [
                {
                    "id": ch.id,
                    "name": ch.name,
                    "grade": ch.grade.value,
                    "edition": ch.edition.value,
                    "subject": ch.subject.value,
                    "description": ch.description,
                    "total_knowledge_points": ch.total_knowledge_points,
                    "estimated_hours": ch.estimated_hours,
                    "level_descriptions": ch.level_descriptions,
                }
                for ch in self._chapters.values()
            ],
            "knowledge_points": [
                {
                    "id": kp.id,
                    "chapter_id": kp.chapter_id,
                    "name": kp.name,
                    "type": kp.type.value,
                    "description": kp.description,
                    "level": kp.level,
                    "mastery_criteria": kp.mastery_criteria.to_dict() if kp.mastery_criteria else None,
                    "teaching_config": kp.teaching_config.to_dict() if kp.teaching_config else None,
                    "dependencies": kp.dependencies,
                }
                for kp in self._knowledge_points.values()
            ]
        }
        
        with open(file_path, 'w', encoding='utf-8') as f:
            json.dump(data, f, ensure_ascii=False, indent=2)
    
    def load_chapters_from_file(self, file_path: str = "data/chapters.json"):
        """从JSON文件加载章节数据"""
        if not os.path.exists(file_path):
            return
        
        with open(file_path, 'r', encoding='utf-8') as f:
            data = json.load(f)
        
        # 加载章节
        for ch_data in data.get("chapters", []):
            chapter = Chapter(
                id=ch_data["id"],
                name=ch_data["name"],
                grade=Grade(ch_data["grade"]),
                edition=Edition(ch_data["edition"]),
                subject=Subject(ch_data["subject"]),
                description=ch_data.get("description"),
                total_knowledge_points=ch_data.get("total_knowledge_points", 0),
                estimated_hours=ch_data.get("estimated_hours"),
                level_descriptions=ch_data.get("level_descriptions", {}),
            )
            self._chapters[chapter.id] = chapter
        
        # 加载知识点
        for kp_data in data.get("knowledge_points", []):
            kp = KnowledgePoint(
                id=kp_data["id"],
                chapter_id=kp_data["chapter_id"],
                name=kp_data["name"],
                type=KnowledgePointType(kp_data["type"]),
                description=kp_data.get("description"),
                level=kp_data.get("level", 0),
                mastery_criteria=MasteryCriteria.from_dict(kp_data["mastery_criteria"]) if kp_data.get("mastery_criteria") else None,
                teaching_config=TeachingConfig.from_dict(kp_data["teaching_config"]) if kp_data.get("teaching_config") else None,
                dependencies=kp_data.get("dependencies", []),
            )
            self._knowledge_points[kp.id] = kp
```

## 十、安全考虑

### 10.1 认证安全

- 管理员密码复杂度要求：至少 8 位，包含大小写字母和数字
- JWT Token 有效期：2 小时
- Refresh Token 有效期：7 天
- 登录失败限制：5 次失败后锁定 30 分钟

### 10.2 操作审计

```python
@dataclass
class AuditLog:
    """操作审计日志"""
    id: int
    admin_id: int          # 操作管理员ID
    action: str            # 操作类型：create/update/delete
    resource_type: str     # 资源类型：chapter/knowledge_point
    resource_id: str       # 资源ID
    old_value: Optional[dict]  # 修改前的值
    new_value: Optional[dict]  # 修改后的值
    ip_address: str        # 操作IP
    created_at: datetime
```

### 10.3 数据备份

- 每次保存配置前自动备份上一版本
- 保留最近 10 个备份版本
- 备份文件存储在 `data/backups/` 目录
- 文件命名格式：`chapters_backup_YYYYMMDD_HHMMSS.json`
