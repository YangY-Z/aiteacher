🧲 电磁感应模板扩展演示

展示模板驱动方案如何处理复杂物理场景

📐 扩展步骤：从简单到复杂

Step 1: 分析场景特征

电磁感应包含：磁铁、线圈、磁场线、电流方向、公式等关键元素

  • 磁铁可以有不同的位置（进入、静止、离开）
  • 线圈匝数可变
  • 磁场线需要动态效果
  • 电流方向符合楞次定律

Step 2: 定义参数结构

LLM只需要输出这些参数：

{ "title": "电磁感应 - 磁铁进入线圈", "magnet_position": "entering", // entering, inside, leaving "coil_turns": 6, // 线圈匝数 "show_magnetic_field": true, // 显示磁场线 "show_current": true, // 显示电流 "show_formula": true // 显示公式框 }

Step 3: 编写模板渲染器

模板引擎根据参数生成精美的SVG：

  • ✅ 自动计算布局和坐标
  • ✅ 添加渐变、阴影、发光效果
  • ✅ 动画效果（磁场线流动、电流指示闪烁）
  • ✅ 物理公式自动匹配

场景1: 磁铁进入线圈 磁通量增加 顺时针电流

物理原理：当磁铁N极接近线圈时，穿过线圈的磁通量增加。根据楞次定律， 感应电流产生的磁场方向与原磁场方向相反，以阻碍磁通量的增加。因此电流方向为顺时针。

场景2: 磁铁静止在线圈内 磁通量不变 无电流

物理原理：当磁铁静止在线圈内时，穿过线圈的磁通量不发生变化。 根据法拉第电磁感应定律 ε = -dΦ/dt，磁通量变化率为零，因此不产生感应电流。

场景3: 磁铁离开线圈 磁通量减少 逆时针电流

物理原理：当磁铁N极远离线圈时，穿过线圈的磁通量减少。根据楞次定律， 感应电流产生的磁场方向与原磁场方向相同，以阻碍磁通量的减少。因此电流方向为逆时针。

📊 三个场景的参数对比

参数 场景1（进入） 场景2（静止） 场景3（离开）
magnet_position "entering" "inside" "leaving"
coil_turns 6 6 6
电流方向 顺时针 无 逆时针
磁通量变化 增加 ↑ 不变 — 减少 ↓

关键点：只需修改一个参数 magnet_position，就能生成完全不同的物理场景图片， 且保证物理原理的准确性！
