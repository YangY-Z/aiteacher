### 2. API文档

#### 年级科目管理API文档

**文件:** `docs/年级科目管理API文档.md`（269行）

**主要内容:**
- 数据模型说明（Grade、Subject、GradeSubject）
- API接口详细说明
- 使用示例和错误处理
- 数据持久化说明

**API端点清单:**

**年级管理:**
```
GET    /api/v1/admin/grades                      # 获取年级列表
POST   /api/v1/admin/grades                      # 创建年级
GET    /api/v1/admin/grades/{id}                 # 获取年级详情
PUT    /api/v1/admin/grades/{id}                 # 更新年级
DELETE /api/v1/admin/grades/{id}                 # 删除年级
GET    /api/v1/admin/grades/{id}/subjects        # 获取年级科目
POST   /api/v1/admin/grades/{id}/subjects        # 添加科目到年级
DELETE /api/v1/admin/grades/{id}/subjects/{sid}  # 从年级移除科目
```

**科目管理:**
```
GET    /api/v1/admin/subjects                    # 获取科目列表
POST   /api/v1/admin/subjects                    # 创建科目
GET    /api/v1/admin/subjects/{id}               # 获取科目详情
PUT    /api/v1/admin/subjects/{id}               # 更新科目
DELETE /api/v1/admin/subjects/{id}               # 删除科目
```

**使用示例:**

```bash
# 1. 创建年级
curl -X POST http://localhost:8000/api/v1/admin/grades \
  -H "Content-Type: application/json" \
  -d '{"name": "初一", "code": "C1", "level": "middle"}'

# 2. 为年级添加科目
curl -X POST http://localhost:8000/api/v1/admin/grades/G_C1/subjects \
  -H "Content-Type: application/json" \
  -d '{"subject_id": "S_MATH", "sort_order": 1}'
```

**快速步骤:**

**添加年级:**
```bash

python scripts/init_grade_subjects.py

# 2. 或通过API创建
curl -X POST http://localhost:8000/api/v1/admin/grades \
  -H "Content-Type: application/json" \
  -d '{"name": "初一", "code": "C1", "level": "middle"}'
```

```bash

  -H "Content-Type: application/json" \
  -d '{"name": "数学", "code": "MATH", "category": "science"}'
```

**配置年级科目:**
```bash

  -H "Content-Type: application/json" \
```
