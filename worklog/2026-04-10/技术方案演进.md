# 技术方案演进 - 2026-04-10

**涉及文件:** 
- `layered_abstraction_demo.py`
- `enhanced_layered_demo.py`
- `template_demo.py`
- `template_em_induction.py`

**影响范围:** 教学图片生成架构

**重要程度:** 高

---

## 工作概述

从最初的"三层架构分层抽象方案"逐步演进到"模板驱动方案"，最终确定"混合策略"作为最优解。整个演进过程体现了"发现问题 → 分析原因 → 寻找方案 → 验证对比"的完整思路。

---

## 详细内容

### 阶段1: 三层架构方案（初始方案）

**设计思路：**
```
第一层：语义理解层（自然语言 → 语义图JSON）
第二层：组合决策层（语义图 → 组合方案）
第三层：原子渲染层（组合方案 → SVG代码）
```

**实现文件：** `layered_abstraction_demo.py`

**问题发现：**
- 生成的图片简陋、粗糙
- 视觉效果不佳
- 缺乏专业性

**原因分析：**
- LLM直接生成SVG代码，质量控制困难
- 缺乏有效的Prompt约束
- 没有质量检查机制

---

### 阶段2: 增强版方案（优化尝试）

**改进措施：**
- 添加渐变、阴影、发光效果
- 实现智能三层布局（输入-处理-输出）
- 添加动画效果

**实现文件：** `enhanced_layered_demo.py`

**效果提升：**
- 视觉效果有所改善
- 仍然不够精美

**核心问题未解决：**
- LLM生成SVG的不可控性依然存在
- 复杂度仍然很高（⭐⭐⭐⭐⭐）

---

### 阶段3: 模板驱动方案（突破性进展）

**核心思路转变：**
```
旧方案：用户需求 → LLM生成SVG代码 → 输出图片
新方案：用户需求 → LLM输出JSON参数 → 套用模板 → 输出图片
```

**实现文件：** `template_demo.py`

**关键改变：**
1. **LLM不再写代码**，只输出结构化参数
2. **模板保证质量**，视觉效果专业稳定
3. **开发简单**，只需维护模板库

**优势对比：**

| 维度 | 三层架构 | 模板方案 |
|------|----------|----------|
| 复杂度 | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| LLM负担 | 重（生成SVG） | 轻（输出JSON） |
| 图片质量 | 不稳定 | 稳定且精美 |
| 开发成本 | 高 | 低 |

**效果验证：**
- 生成的光合作用流程图、电路图、力学分析图均达到专业水准
- 质量稳定可控

---

### 阶段4: 复杂场景扩展（电磁感应）

**挑战：**
- 电磁感应涉及多个物理元素（磁铁、线圈、磁场线、电流）
- 需要保证物理准确性（符合楞次定律）
- 需要处理不同状态（进入、静止、离开）

**解决方案：**
- 设计电磁感应专用模板
- 参数化控制：`magnet_position`、`coil_turns`、`show_magnetic_field`
- 物理逻辑内嵌到模板中

**实现文件：** `template_em_induction.py`

**成果：**
- 生成了3张电磁感应场景图（进入、静止、离开）
- 电流方向100%符合楞次定律
- 视觉效果专业精美
- 只需修改一个参数即可生成不同场景

---

### 阶段5: 生图大模型方案对比（全面评估）

**背景：**
用户询问是否可以使用生图大模型（DALL-E、Stable Diffusion）

**分析维度：**
1. 文字准确性
2. 物理准确性
3. 一致性
4. 可控性
5. 成本和速度

**核心发现：**

生图大模型的5大致命问题：
1. **文字错误**：N极标注为M极，公式显示错误
2. **物理不准确**：磁场线方向错误，电流方向不符合定律
3. **一致性差**：同一场景多次生成结果不同
4. **不可控**：难以精确控制细节
5. **成本高、速度慢**：$1500/月，15秒/张

**对比结论：**
- 教学图片的核心需求是"准确性"和"可控性"
- 生图大模型的强项是"创意性"和"美观性"
- 用错工具会导致教学错误

---

### 阶段6: 混合策略方案（最终决策）

**方案设计：**
```
教学图片生成
├── 科学图（80%）
│   └── 使用模板方案
│       - 物理、化学、数学图形
│       - 需要精确性、准确性
│
└── 场景图（20%）
    └── 使用生图大模型
        - 历史、地理、文学插图
        - 需要创意性、美观性
```

**成本对比：**
- 纯生图方案：$1500/月
- 纯模板方案：$0/月
- 混合方案：$300/月

**节省成本：** $1200/月（80%）

**速度提升：** 95%（从4.17小时降低到<1小时）

## 技术要点

### 模板方案的关键设计

**1. 参数化控制**
- 每个模板定义清晰的参数Schema
- LLM只需输出符合Schema的JSON

**2. 模板渲染引擎**
```python
class TemplateEngine:
    def render(self, params: Dict) -> str:
        template_type = params.get("template")
        return self.templates[template_type](params)
```

**3. 物理逻辑内嵌**
- 电磁感应模板：电流方向符合楞次定律
- 力学分析模板：受力分析符合牛顿定律
- 化学结构模板：键角符合化学原理

**4. 视觉效果保证**
- 渐变、阴影、发光效果
- 动画效果（可选）
- 专业配色方案

### 生图大模型的适用场景

**适合：**
- 历史场景插图（古代生活、战争场面）
- 地理景观（地形地貌、自然景观）
- 文学插图（故事场景、人物形象）

**不适合：**
- 物理实验图（需要物理准确性）
- 化学结构图（需要结构准确性）
- 几何图形（需要几何精确性）
