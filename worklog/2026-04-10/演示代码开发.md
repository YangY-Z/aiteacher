### 3. 模板驱动方案演示（核心突破）

**文件:** `template_demo.py`

**功能:**
- 实现模板渲染引擎
- 定义3种模板：流程图、电路图、力学分析
- 展示LLM只需输出JSON参数

**输出:**
- `demo_template_1.svg`（光合作用流程图）
- `demo_template_2.svg`（串联电路图）
- `demo_template_3.svg`（力学分析图）

**效果评价:**
- ✅ 质量稳定精美
- ✅ 复杂度大幅降低
- ✅ 可控性强

**代码规模:** 约300行

**核心设计:**

```python
class TemplateEngine:
    """模板渲染引擎"""
    
    def __init__(self):
        self.templates = {
            "process_flow": self._process_flow_template,
            "circuit_simple": self._circuit_template,
            "mechanics_force": self._mechanics_template
        }
    
    def render(self, params: Dict) -> str:
        """根据参数选择模板并渲染"""
        template_type = params.get("template")
        return self.templates[template_type](params)
```

**LLM输出示例:**
```json
{
  "template": "process_flow",
  "title": "光合作用过程",
  "steps": ["光能吸收", "水分解", "氧气释放", "有机物合成"],
  "colors": ["#4CAF50", "#2196F3", "#FF9800", "#E91E63"]
}
```

```python
def _generate_current_indicator(self, magnet_position: str, show: bool):
    """根据磁铁运动方向决定电流方向（符合楞次定律）"""
    if magnet_position == "entering":
        direction = "顺时针"  # 磁通量增加，阻碍增加
    elif magnet_position == "leaving":
        direction = "逆时针"  # 磁通量减少，阻碍减少
    else:  # inside
        direction = "无"  # 磁通量不变，无感应电流
```

## 技术要点

### 1. 模板渲染引擎设计

**核心模式：** 策略模式

```python
class TemplateEngine:
    def __init__(self):
        self.templates = {
            "template_type": self._template_method
        }
    
    def render(self, params):
        return self.templates[params["template"]](params)
```

**优势：**
- 易于扩展新模板
- 代码结构清晰
- 符合开闭原则

### 2. SVG 生成技术

**关键技术：**
- 渐变（linearGradient）
- 滤镜（filter）
- 动画（CSS animation）
- 文本渲染（text元素）
- 路径绘制（path元素）

**示例：**
```xml
<!-- 渐变 -->
<defs>
    <linearGradient id="grad" x1="0%" y1="0%" x2="0%" y2="100%">
        <stop offset="0%" style="stop-color:#4CAF50"/>
        <stop offset="100%" style="stop-color:#2E7D32"/>
    </linearGradient>
</defs>

<!-- 应用渐变 -->
<rect fill="url(#grad)" />
```

### 3. 参数化控制

**JSON Schema 设计：**

```json
{
  "template": "string (required)",
  "title": "string (optional)",
  "steps": "array of strings (required for process_flow)",
  "colors": "array of colors (optional)",
  "magnet_position": "enum: entering|inside|leaving (required for em_induction)",
  "coil_turns": "integer (optional)"
}
```
