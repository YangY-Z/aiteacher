```dockerfile

    ffmpeg \
    libcairo2-dev \
    libpango1.0-dev

RUN pip install manim==0.18.0

ENV MANIM_CONFIG_FILE=/tmp/manim/manim.cfg
```

## 潜在风险和注意事项

### 1. 性能风险
- **问题**: 高并发场景下，OpenSandbox可能成为瓶颈
- **缓解**: 
  - 添加异步任务队列（Celery）
  - 实现资源池
  - 使用Firecracker运行时

### 2. 资源消耗
- **问题**: 每个沙箱占用100-200MB内存
- **缓解**:
  - 设置合理的超时时间
  - 自动清理过期沙箱
  - 监控资源使用情况

### 3. 依赖风险
- **问题**: OpenSandbox是相对新的项目
- **缓解**:
  - 关注官方更新
  - 准备降级方案（直接Docker）
  - 定期测试稳定性

### 使用示例1: 生成一次函数动画

```python
from app.services.tools.animation_generator import animation_generator
import asyncio


async def main():
  result = await animation_generator.generate_animation(
    animation_type="linear_function",
    params={"k": 2, "b": 1},
    trace_id="demo-001"
  )

  print(f"视频URL: {result['video_url']}")
  print(f"时长: {result['duration']}秒")
  print(f"是否缓存: {result['cached']}")


asyncio.run(main())
```
