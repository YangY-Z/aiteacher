## 🏗️ 架构总结

### Agent 工作流程

```
1. 前端调用 /api/v1/improvement/agent/run
   ↓
2. Agent 初始化会话
   ↓
3. Agent 进入 tool_use 循环（最多 20 轮）
   ├─ 调用 LLM，请求工具调用
   ├─ LLM 返回 JSON：{action, tool_name, tool_input}
   ├─ Agent 执行工具
   ├─ 工具结果反馈给 LLM
   └─ 重复直到 LLM 决定结束
   ↓
4. 返回最终会话状态
```

### 核心工具（5 个）

| 工具 | 功能 | 状态 |
|------|------|------|
| diagnose_student | 诊断薄弱知识点 | ✅ |
| clarify_student | 澄清知识缺口 | ✅ |
| generate_learning_plan | 生成学习方案 | ✅ |
| teach_knowledge_point | 生成教学内容 | ✅ |
| evaluate_quiz | 评估小测答题 | ✅ |

### 可扩展工具（4 个）

## 🚀 快速开始

### 启动服务
```bash
# 后端
cd ai-teacher-backend
python3 run.py

# 前端
cd ai-teacher-frontend
npm run dev
```

### 测试 Agent 端点
```bash
curl -X POST http://localhost:8008/api/v1/improvement/agent/run \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "course_id": "MATH_JUNIOR_01",
    "score_input": {
      "exam_name": "一元二次函数",
      "score": 60,
      "total_score": 100,
      "available_time": 30,
      "difficulty": "normal",
      "foundation": "average"
```
